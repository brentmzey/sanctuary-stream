# 🚀 Sanctuary Stream - One-Command Development

**Start everything with one command!**

---

## Super Quick Start

```bash
# Option 1: tmux (Recommended)
npm run dev:tmux

# Option 2: iTerm2 (macOS)
npm run dev:iterm

# Option 3: Simple (Works everywhere)
npm run dev:simple
```

---

## What Happens

All three commands start:
1. 🗄️  **PocketBase** - Database at http://127.0.0.1:8090
2. ⚙️  **Bridge** - Connects to OBS
3. 🌐 **Web App** - Control panel at http://localhost:5173

---

## Which Should I Use?

| Command | Best For | Requires |
|---------|----------|----------|
| `npm run dev:tmux` | Linux/macOS power users | tmux |
| `npm run dev:iterm` | macOS users (prettiest!) | iTerm2 |
| `npm run dev:simple` | Everyone (simplest) | Just bash |

---

## First Time Setup

```bash
# 1. Install dependencies
npm install

# 2. Download PocketBase
# Visit: https://pocketbase.io/docs/
# Extract to: pocketbase/local/pocketbase

# 3. Start everything
npm run dev:tmux
```
